## Descripcion Interfaz 1
Esta interfaz permite registrar visitantes de una institución. Se capturan datos como nombre, cédula, teléfono, motivo de visita, persona a visitar y tipo de visitante. También permite seleccionar el documento entregado y visualizar todos los registros en una tabla.

## Descripcion Interfaz 2
Esta interfaz permite registrar productos en un inventario. El usuario puede ingresar información como código, nombre, categoría, precio, cantidad y proveedor. También permite seleccionar el estado del producto y marcar opciones adicionales como si es perecedero o importado, Los registros se muestran en una tabla.

## Descripcion Interfaz 3
Esta interfaz simula un perfil de estudiante donde se almacenan datos personales del alumno como matrícula, nombre, carrera y contacto. Este, muestra las materias inscritas y un área para observaciones.
